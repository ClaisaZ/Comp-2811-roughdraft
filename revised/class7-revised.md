# Comms / UX / UI Design Redesign Project - Class 7

This week we focused on applying usability heuristics, defining our visual design choices, and refining the overall direction we are taking in our redesign.

## Usability Heuristics Applied

We selected and focused on three key heuristics, that will shape and guide our redesign.

**1. Aesthetics and Minimalist Design**

The original website felt cluttered and lacked clear visual hierarchy, making it difficult for users to focus on their intended purpose.

To tackle this problem the redesign will simplify the layouy by removing unnecessary elements and organizing content into sections. We will also try to increase whitespace, alignment issues, consistent image sizing, and a more fitting color scheme.

These changes will hopefully bring in a more user-friendly experience.

**2. Consistency and Standards**

The original website had inconsistent layouts, navigation, and visual styles, which made the site feel disconnected.

In the redesign we will improve the navigation, button styles, and layout structure to improve the consistency and overall standardizing the application to meet our usability heuristics.

This will hopefully let users navigate the website more easily, without having to learn different user flows across differnt pages

**3. Match Between System and the Real World**

The original website did not meet user expectations on how a card shop should behave and how it should be organized.

The redesign structures the website similarly to how a physical store organizes products, creating familiar sections. We also want to improve the event section and highlight important details such as the event type, dates, formats, and registration, which has been highly lacking in the original design.

This will create a more intuitive browsing experience that might be able to match user expectaions.

## Typography

Typography was selected to improve readability and maintain a modern visual style, while keeping the appearance in mind as well.

**Display/Heading Font**

Montserrat or Roboto
Used for headings and section titles due to their strong, modern appearance.

**Body Font**

Open Sans
Used for body text because of its clarity and high readability.

These fonts are a strong standard pair for creating a clear visual hierarchy while maintaining consistency across the website.

## Key Insight

This week showed how usability heuristics can influence design decisions. By applying principles like minimalism, consistency, and real-world alignment, the redesign became more intentional rather than based on assumptions.

Choosing colors and typography also reinforced that visual design is not just about appearance, but about improving usability, readability, and overall user experience.
