# Comms / UX / UI Design Redesign Project — Class 9

**High-Fidelity Wireframes**

This week focused on developing high-fidelity wireframes by applying finalized color schemes, typography, and visual styling to the redesign.

This week we also did in-person user testing and a review with Chris, which helped us get direct feedback and see what we needed to improve.

These designs build on previous wireframes and incorporate improvements based on user testing results.

## Strengths Identified

The high-fidelity wireframes showed many strong improvements:

- Clear and consistent layout structure
- Effective use of whitespace to improve readability
- Intuitive navigation that supports user flow
- Strong color choices that align with branding and accessibility
- Well chosen typography that enhances visual hierarchy

Additionally, confirmation windows were significantly improved from earlier iterations, providing clearer feedback and better communication to users.

## Areas for Improvement

Based on feedback, several areas of improvements were identified:

**Alignment:**
Elements such as buttons and content blocks need to be consistenly aligned to create a more finished and professional layout.

**Spacing Consistency**:
Buttons and components should maintain equal spacing on all sides to improve visual balance.

**Button Grouping**:
Related buttons should be placed closer together to reflect their relationship and improve usability.

**Button Consistency**:
Buttons should follow consistent styles in terms of size, spacing, design, and color.

**Confirmation Windows Design:**
Confirmation windows should be slightly smaller to improve whitespace and reduce visual clutter.

## Design Improvments

To improve and fix these issues our team will try to apply these changes:

- Standardize alignment across all elements using consistent layout grids
- Apply uniform spacing between components, sections, images
- Group related actions together or simply try to make them closer to improve usability
- Ensure all buttons follow a consistent design structure
- Finishing touches on confirmation windows, white space and balance.

## Key Insight

This week showed that even when the overall design is strong, small details such as alignment, spacing, and consistency have a big impact on the final user experience. Feedback from in-person testing and review showed areas that were not immediately obvious during design.

By fixing these details, the redesign becomes more polished, intuitive, and visually consistent, showing us the importance of having good structure in the process.
