# Comms / UX / UI Design Redesign Project - Class 5

**User Flows**

This week focused on understanding and creating user flows to support the redesign process.

User flows are diagrams that outline the steps a user takes to achieve a specific goal within a website or application.

They help visualize how users interact with a system and allow designers to identify potential problems in the experience.

## Example User Story

User flows are often bason on user stories.

Example: As a collector, I want to see the latest deals so that I can purchase items before they sell out.

## Example User Flow

Based on this user Story, we were able to create a user flow.

Start -> Open Menu -> Clicks "Latest Deals" -> View Products -> Select Product -> Add to card -> Enter payment information -> Complete purchase

This flow represents a typical path a user might take when interacting with a website.

## Application to the Redesign

Creating user flows helped the team understand how users move through the Magic Stronghold website and where improvements can be made.

It also ensured that important actions, such as browsing products or searching for events, are simple and user friendly.
