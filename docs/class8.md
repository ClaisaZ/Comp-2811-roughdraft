Part 1: Three Usability Heuristics

1. Aesthetic and Minimalist Design
The original site felt cluttered with way too much information and poor visual hierarchy. We plan to simplify the layout by reducing unnecessary elements ( random logo, buttons, large banners) and organizing content into more clear sections. We definitely need to add more whitespace, improve alignment, change color scheme, and use consistent image sizes. This will make the site easier to view and prevents users from feeling overwhelmed.

2. Consistency and Standards
The original website had inconsistent layouts, navigation styles, and product names which made the interface feel confusing and cluttered and slightly disconnected ( top menu and left side bar spacing). In the redesign, we plan to change the navigation bar, button styles, spacing, and the usage of white-space. For example, all product categories ( singles, sealed, accessories ) are now placed in specific navigation section. This helps users quickly understand how to move through the site without relearning different layouts.

3. Match Between System and the Real World
The original design doesn’t fully match how users expect a card shop to be organized, which makes navigation less fluid. While categories and sections exist, they are not always shown in a way that people browse in a real store. In our redesign, we want to structure the site more like a physical shop by clearly organizing categories such as singles, sealed products, and accessories. We also plan to make the tournaments a bigger focus by improving the Upcoming Events section with clearer information like dates, formats, and sign-up options just like our user flow.

Part 2: Color Palette (RGB)

The color palette uses a triadic color scheme to create visual balance while still providing contrast for important elements. This approach helps maintain a cohesive design while making key actions stand out.

Primary Color (Most Used):
RGB (11, 46, 162)
A deep blue used as the main background color. This reflects the gaming/card shop aesthetic, we also chose not to stray to far from the original color.

Secondary Color (Less Used):
RGB (59, 85, 226)
A slightly lighter blue used for sections such as cards and categories. This adds visual variety/hierarchy while still fitting within the theme. Debating whether to use a darker blue rgb(38, 67, 164) than the primary color. We will test more for our final product.

Highlight Color (Accent):
RGB (255, 155, 39)
A orange color is used for buttons, links, and important actions like register/login, card, checkout, etc. This color draws attention without clashing, since orange is a complimentary color to blue

It is also important to note that all the words and fonts in the redesign will either be black or white and that when a clickable event is hovered the color will switch showing the user a visual cue.

We are also debating whether to have the default browser be dark-themed.
